02.0M — SHALLOWS / 淺灘

新手上路

還沒下水?先搞懂這些再談考證。點開你想看的。

BASICS🫧 下水前必懂 8 件事＋
🫁

水肺 vs 自由潛

一個背氣瓶待水下、一個閉氣下潛。先決定玩哪種。

👂

平壓(耳壓)

耳朵痛就捏鼻鼓氣;痛就停、上升一點再試,別硬撐。

🌬

絕不憋氣

水肺鐵律:永遠正常呼吸,憋氣上升可能傷肺。

👥

潛伴制度

永遠不獨潛,潛伴互相照應是安全核心。

⬆️

慢上升+安全停留

上升要慢,5m 停留 3 分鐘,防減壓病。

🤧

身體條件

感冒鼻塞別潛;潛完 18–24h 內別搭飛機。

🥽

裝備入門

面鏡/蛙鞋/BCD/調節器/電腦錶。初期租借就好。

🏁

怎麼開始

先體驗潛水(DSD)試水溫,喜歡再考 OWD。

05
05.0M — DEEPER / 深一點

潛水實操

已經有證、想自己揪團或規劃潛水?下水前先想清楚。

🧭

評估自己能力

證照等級、近期頻率、最大深度、氣量與中性浮力。誠實面對。

⚖️

對比潛點難度

把深度/洋流/入水方式/能見度對照你的能力,不逞強。

🔍

潛點分析

查海況潮汐、洋流、入水出水點、最大深度與無減壓時間。

📋

潛水計畫

氣量規劃、隊形、失散處理、水面支援、最近減壓艙。
